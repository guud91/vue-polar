Vue Polar examples

This page contains a few examples of Vue Polar complete with code samples. Add the following lines to your <head> section to start using Vue Polar: 

<script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
<script src="https://unpkg.com/vue-polar@latest/dist/vuePolar.umd.min.js"></script>
  • Documentation and source code on Github: github.com/snirp/vue-polar
  • NPM package: npmjs.com/package/vue-polar
